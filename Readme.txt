English:

Instructions:
1 - In the game folder, copy the contents of the "Files for creation" folder and the contents of the game folder into the "game" folder.

2 - Edit the "autorun.cmd" file (with a text editor) and paste the game executable.
Example:
DIR=game
CMD="Launcher.exe"
If the executable is not in the game folder but in a subfolder, for example, "game\bin", the path will be:
DIR=game\bin
CMD="Launcher.exe"

Save the edited file.

Then, inside the game folder, there should be the following folders and files:
drive_c
game
autorun.cmd

3 - Run the game to wsquashfs.bat file, selecting the game folder we edited earlier.

4 - Give the same name as the game folder.
Example:
The game folder is called Tekken 3, the file should be Tekken 3.wsquashfs.

5 - Wait for the process to finish, which will depend on the size of the game in megabytes or gigabytes and/or the performance of your PC.


Italian:

Istruzioni: 
1 - Nella cartella del gioco, copiare il contenuto presente nella cartella "Files for creation" e il contenuto della cartella del gioco dentro alla cartella "game".

2 - Editare il file "autorun.cmd" ( con un editor di testo) e inserire l'eseguibile del gioco
Esempio: 
DIR=game
CMD="Launcher.exe"
Se l'exe non è nella cartella game ma in una sottocartella, per esempio "game\bin", il percorso sarà:
DIR=game\bin
CMD="Launcher.exe"

Salvare il file editato.

Quindi dentro alla cartella del gioco ci dovranno essere le seguenti cartelle e file:
drive_c
game
autorun.cmd

3 - Lancia il file game to wsquashfs.bat, seleziona la cartella del gioco che abbiamo modificato prima.

4 - dare il nome uguale alla cartella del gioco.
Esmpio:
La cartella del gioco si chiama Tekken 3, il file dovrà essere Tekken 3.wsquashfs.

5 - Aspettare la fine del processo che dipenderà dalla grandezza del gioco in termine di mega o giga byte e/o dalla potenza del pc.
